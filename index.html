Trilha de Programação

O novo ensino médio inclui a disciplina de Matemática II (Programação) para os alunos que escolheram a trilha de exatas.

Durante o trimestre os alunos produziram seus protótipos de site e após isso construíram seus sites utilizando as linguagens de programação HTML e CSS e realizaram um curso na plataforma ALURA.

Explorando a criatividade, os alunos criaram sites com temas variados.
